# Archive - Documents internes

Ce répertoire contient les documents internes qui ne doivent pas être publiés
dans la documentation publique ou le wiki GitHub.

## Contenu typique

- Notes de conception (design docs)
- Spécifications techniques internes
- Brouillons d'architecture
- Documents de travail non-finalisés
- Prototypes et preuves de concept

## Règles

- Les documents ici ne sont pas versionnés pour la publication
- Ils peuvent être supprimés ou déplacés vers la documentation publique
  une fois finalisés
- Ce dossier est ignoré par la génération de la documentation publique
