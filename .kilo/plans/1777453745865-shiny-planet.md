## 🎯 Fonctionnalités à Implémenter

### P0 (Critique)

**P0-1: Pipeline Tracking System**
- UUID pipeline généré automatiquement
- Suivi d'état global (pending → running → completed/failed)
- Tracking par step: status + started_at + finished_at + retries + error
- Storage abstrait (Redis, Memory, extensible DB)
- Auto-detect broker → storage adapté
- API: `get_pipeline_status(pipeline_id)`, `list_pipelines(limit)`

**P0-2: Hooks & Events**
- Callbacks: `on_pipeline_start`, `on_step_start`, `on_step_complete`, `on_pipeline_complete`, `on_error`
- HookManager: register/unregister/dispatch async non-blocking
- Prêt pour WebSocket (bridge vers chanx)
- Injection dans PipelineMiddleware

**P0-3: Parallel Map Helper**
- `parallel_map(task, iterable, **kwargs)` — crée un groupe de tâches parallèles
- Équivalent à `group(*[task.kiq(item) for item in iterable])`

**P0-4: Chunked Map Helper**
- `chunked_map(task, items, chunk_size=None, max_concurrency=None, auto=False)`
- Partitionnement automatique + exécution parallèle limitée
- Concurrency configurable ou auto-detect (CPU, broker type)

### P1 (Important)

**P1-1: Conditions Dynamiques**
- `Pipeline.if(condition, task)` ou `maybe(task, condition)`
- Execution conditionnelle depuis résultat précédent

**P1-2: Retry par Node (fine-grained)**
- Via SmartRetryMiddleware taskiq (labels standards)
- `.with_options(retries=3, retry_delay=10)` par step
- Propagation des options retry/timeout

**P1-3: Logging Structuré**
- Format JSON (production) + lisible (debug)
- Configurable via `TASKIQ_PIPELINES_LOG_FORMAT`
- Champs: timestamp, pipeline_id, step, task, event, duration, error

**P1-4: Error Handling Avancé**
- `Pipeline.with_options(fail_fast=True|False)`
- `Pipeline.with_options(continue_on_error=True|False)`
- Comportement par défaut: fail_fast=True

**P1-5: Scheduling with APScheduler**
- `PipelineScheduler(broker, job_store_url=None)`
- `.schedule(pipeline, cron="*/5 * * * *", args=...)`
- `.schedule_at(pipeline, datetime, args=...)`
- `.schedule_interval(pipeline, hours=1, args=...)`
- Job store: SQLite par défaut (fichier local), extensible (Redis, PostgreSQL)
- Persistance: jobs survivent au restart
- Misfire handling, timezone, jitter support

### P2 (Nice to have)

**P2-1: Step de Branchement (Branch)**
- `Pipeline.branch([steps_a], [steps_b], ...)`
- Exécute branches indépendantes en parallèle
- Attend toutes, puis continue

**P2-2: Step de Réduction (Reduce)**
- `Pipeline.reduce(aggregator_task, initial=None)`
- Agrégation cumulative sur liste

**P2-3: Propagation Améliorée des Résultats**
- `PipelineContext`: accès résultats précédents par task_id
- `context.get_result(task_id)` depuis result_backend

**P2-4: Timeout par Step**
- `.with_options(timeout=30)` par step
- Propagation via labels

**Interface `PipelineStorage` (ABC):**

```python
from abc import ABC, abstractmethod
from datetime import datetime
from typing import Any

class PipelineStorage(ABC):
    """Storage interface for pipeline tracking state."""
    
    @abstractmethod
    async def create_pipeline(self, pipeline_id: str, total_steps: int) -> None: ...
    
    @abstractmethod
    async def start_pipeline(self, pipeline_id: str) -> None: ...
    
    @abstractmethod
    async def complete_pipeline(self, pipeline_id: str, result: Any) -> None: ...
    
    @abstractmethod
    async def fail_pipeline(self, pipeline_id: str, error: str) -> None: ...
    
    @abstractmethod
    async def start_step(self, pipeline_id: str, step_index: int, task_id: str, task_name: str) -> None: ...
    
    @abstractmethod
    async def complete_step(self, pipeline_id: str, step_index: int) -> None: ...
    
    @abstractmethod
    async def fail_step(self, pipeline_id: str, step_index: int, error: str) -> None: ...
    
    @abstractmethod
    async def get_pipeline_status(self, pipeline_id: str) -> PipelineStatusInfo | None: ...
    
    @abstractmethod
    async def list_pipelines(self, limit: int = 10) -> list[PipelineStatusInfo]: ...
    
    @abstractmethod
    async def cleanup_old(self, ttl_seconds: int = 3600) -> int: ...
```

**Redis Implementation:**
- Keys: `pipe:{pipeline_id}` → hash (status, created_at, started_at, finished_at, total_steps, current_step, result, error)
- Keys: `pipe:{pipeline_id}:steps` → list JSON (étapes)

- Pipeline HSet + Steps List + TTL

- Dict: `{pipeline_id: PipelineStatusInfo}`

**HookManager:**

```python
class HookManager:
    def __init__(self):
        self._callbacks: dict[str, list[Callable]] = {}
        self._lock = asyncio.Lock()
    
    def register(self, event_type: str, callback: Callable[[PipelineEvent], None | Coroutine]) -> None: ...
    def unregister(self, event_type: str, callback: Callable) -> None: ...
    async def dispatch(self, event: PipelineEvent) -> None:
        async with self._lock:
            for cb in self._callbacks.get(event.__class__.__name__, []):
                asyncio.create_task(self._safe_call(cb, event))
    
    async def _safe_call(self, cb, event):
        try:
            await cb(event) if iscoroutine(cb) else cb(event)
        except Exception:
            logger.exception("Hook callback failed")
```

```python
class WebSocketHookBridge:
    def __init__(self, hook_manager: HookManager, channel_layer: ChannelLayer):
```

```python
async def kiq(self, *args, **kwargs) -> AsyncTaskiqTask:
    if not self.steps:
        raise ValueError("Pipeline empty")
    
    # Generate ID + init tracking
    if self.tracking_enabled:
        self.pipeline_id = str(uuid4())
        await self.tracking_manager.initiate(self.pipeline_id, len(self.steps))
        await self.hook_manager.dispatch(PipelineStartEvent(pipeline_id=self.pipeline_id, pipeline=self))
    
    self._update_task_ids()
    
    # Enrichir labels: PIPELINE_ID, PIPELINE_DATA, CURRENT_STEP
    step = self.steps[0]
    parsed = parse_step(step.step_type, step.step_data)
    kicker = AsyncKicker(parsed.task_name, broker=self.broker, labels=parsed.labels)
        .with_task_id(step.task_id)
        .with_labels(
            _pipe_id=self.pipeline_id,
            _pipe_data=self.dumpb(),
            _pipe_current_step=0,
            _step_retries=str(self.options.default_retries) if self.options.default_retries else None,
            _step_timeout=str(self.options.default_timeout) if self.options.default_timeout else None,
        )
    
    task = await kicker.kiq(*args, **kwargs)
    task.task_id = self.steps[-1].task_id
    return task
```

### 4. Middleware Enhancements

```python
class PipelineMiddleware(TaskiqMiddleware):
    def __init__(
        self,
        tracking_manager: PipelineTrackingManager | None = None,
        hook_manager: HookManager | None = None,
    ):
        self.tracking_manager = tracking_manager
        self.hook_manager = hook_manager
        super().__init__()
    
    async def post_save(self, message: TaskiqMessage, result: TaskiqResult):
        if not self._is_pipeline_message(message):
            return
        
        pipeline_id = message.labels["_pipe_id"]
        current_step = int(message.labels["_pipe_current_step"])
        steps_data = self._load_steps(message)
        
        # Hook: step_start
        if self.hook_manager:
            await self.hook_manager.dispatch(StepStartEvent(
                pipeline_id=pipeline_id,
                step_index=current_step,
                task_name=steps_data[current_step].task_name,
                task_id=steps_data[current_step].task_id,
            ))
        
        # Tracking: mark step start
        if self.tracking_manager:
            await self.tracking_manager.mark_step_start(
                pipeline_id, current_step,
                steps_data[current_step].task_id,
                steps_data[current_step].task_name,
            )
        
        # Logging
        logger.info(
            f"[{pipeline_id}][STEP {current_step}] START {steps_data[current_step].task_name}",
            extra={"pipeline_id": pipeline_id, "step": current_step, "task": steps_data[current_step].task_name, "event": "step_start"}
        )
        
        # Execute next step
        try:
            next_step = parse_step(...)
            await next_step.act(broker=self.broker, ...)
            
            # Hook: step_complete
            if self.hook_manager:
                await self.hook_manager.dispatch(StepCompleteEvent(
                    pipeline_id=pipeline_id,
                    step_index=current_step,
                    task_name=...,
                    task_id=...,
                    result=result.return_value,
                ))
            
            # Tracking: complete
            if self.tracking_manager:
                await self.tracking_manager.mark_step_complete(pipeline_id, current_step)
            
            logger.info(f"[{pipeline_id}][STEP {current_step}] DONE", extra={...})
            
        except Exception as exc:
            # Tracking: fail
            if self.tracking_manager:
                await self.tracking_manager.mark_step_failed(pipeline_id, current_step, str(exc))
            # Hook: error
            if self.hook_manager:
                await self.hook_manager.dispatch(StepErrorEvent(
                    pipeline_id=pipeline_id,
                    step_index=current_step,
                    task_name=...,
                    task_id=...,
                    error=exc,
                ))
            # Re-raise for middleware chain
            raise
    
    async def on_error(self, message, result, exception):
        # Similar: dispatch error event, fail pipeline tracking
        ...
```

```python
class BranchStep(pydantic.BaseModel, AbstractStep, step_name="branch"):
    branches: list[list[DumpedStep]]  # chaque branche = liste de steps
    
    async def act(self, broker, step_number, parent_task_id, task_id, pipe_data, result):
        # Lancer toutes les branches en parallèle
        async def run_branch(branch_steps):
            # Créer sous-pipeline temporaire avec ces steps
            sub_pipeline = Pipeline(broker)
            sub_pipeline.steps = branch_steps
            # Exécuter avec même input (result.return_value)
            return await sub_pipeline.kiq(result.return_value)
        
        tasks = [run_branch(branch) for branch in self.branches]
        branch_results = await asyncio.gather(*tasks)
        
        # Combiner résultats selon stratégie (par défaut: liste)
        combined = combine_branch_results(branch_results)
        
        # Marquer step完成, passer combined à l'étape suivante
        await self._proceed_to_next(...)
```

```python
class ReduceStep(pydantic.BaseModel, AbstractStep, step_name="reduce"):
    task: SequentialStep  # tâche d'agrégation (accumulator, item) -> new_accumulator
    initial: Any | None = None
    
    async def act(self, broker, step_number, parent_task_id, task_id, pipe_data, result):
        items = result.return_value  # liste itérable
        accumulator = self.initial
        
        for idx, item in enumerate(items):
            # Exécuter task avec accumulator + item
            # Le task doit prendre 2 args: (accumulator, item) ou kwargs spécifiés
            result_step = await self.task.act_with_input(broker, step_number, ...)
            accumulator = result_step.return_value
        
        # Passer accumulator final à next step
        await self._proceed_to_next(accumulator)
```

### 7. Parallel Utilities (`utils/parallel.py`)

```python
from typing import Iterable, Callable, TypeVar
from . import Pipeline

T = TypeVar("T")
R = TypeVar("R")

def parallel_map(
    task: AsyncKicker[[T], R] | AsyncTaskiqDecoratedTask[[T], R],
    iterable: Iterable[T],
    **task_kwargs: Any,
) -> Pipeline:
    """
    Crée un pipeline exécutant `task` sur chaque élément en parallèle.
    Équivalent à group(*[task.kiq(item, **kwargs) for item in iterable]).
    """
    pipeline = Pipeline(broker=task.broker)
    for item in iterable:
        pipeline = pipeline.call_next(task, **task_kwargs)  # param_name=None si item premier arg
        # Note: besoin de clonage du task? Implementation detail
    return pipeline

def chunked_map(
    task: AsyncKicker[[list[T]], R],
    items: list[T],
    chunk_size: int | None = None,
    max_concurrency: int | None = None,
    auto_concurrency: bool = False,
    **task_kwargs: Any,
) -> Pipeline:
    """
    Chunk items, process chunks in parallel with concurrency limit.
    auto_concurrency = True → détecte CPU count, broker capabilities.
    """
    if chunk_size is None:
        chunk_size = auto_chunk_size(items)  # heuristique: len(items)//cpu_count
    
    chunks = [items[i:i+chunk_size] for i in range(0, len(items), chunk_size)]
    
    if auto_concurrency:
        max_concurrency = detect_max_parallelism(task.broker)
    elif max_concurrency is None:
        max_concurrency = len(chunks)  # tout parallèle
    
    # Semaphore pour limiter
    semaphore = asyncio.Semaphore(max_concurrency)
    
    async def process_chunk(chunk):
        async with semaphore:
            return await task.kiq(chunk, **task_kwargs)
    
    # Créer pipeline
    pipeline = Pipeline(task.broker)
    for chunk in chunks:
        pipeline = pipeline.call_next(process_chunk)
    
    return pipeline
```

### 8. Scheduler (APScheduler Integration)

**Structure:**

```python
# scheduling/scheduler.py
class PipelineScheduler:
    def __init__(
        self,
        broker: AsyncBroker,
        scheduler: AsyncIOScheduler | None = None,
        job_store_url: str | None = None,  # default: sqlite:///./scheduler_jobs.db
    ):
        self.broker = broker
        self.scheduler = scheduler or AsyncIOScheduler()
        self._configure_job_store(job_store_url or "sqlite:///./scheduler_jobs.db")
    
    def _configure_job_store(self, url: str):
        if url.startswith("sqlite"):
            from apscheduler.jobstores.sqlalchemy import SQLAlchemyJobStore
            self.scheduler.add_jobstore(SQLAlchemyJobStore(url=url), alias="default")
        elif url.startswith("redis"):
            from apscheduler.jobstores.redis import RedisJobStore
            self.scheduler.add_jobstore(RedisJobStore(host="localhost", port=6379), alias="default")
        elif url.startswith(("postgresql", "mysql")):
            from apscheduler.jobstores.sqlalchemy import SQLAlchemyJobStore
            self.scheduler.add_jobstore(SQLAlchemyJobStore(url=url), alias="default")
    
    async def schedule(
        self,
        pipeline: Pipeline,
        cron: str,
        args: tuple = (),
        kwargs: dict | None = None,
        timezone: str = "UTC",
        misfire_grace_time: int = 300,
        jitter: int | None = None,
    ) -> str:
        trigger = CronTrigger(cron, timezone=timezone, jitter=jitter)
        job = self.scheduler.add_job(
            func=self._run_pipeline,
            trigger=trigger,
            args=[pipeline, args, kwargs or {}],
            misfire_grace_time=misfire_grace_time,
        )
        return job.id
    
    async def schedule_at(self, pipeline: Pipeline, run_at: datetime, ...) -> str: ...
    async def schedule_interval(self, pipeline: Pipeline, ...) -> str: ...
    
    async def _run_pipeline(self, pipeline: Pipeline, args: tuple, kwargs: dict):
        return await pipeline.kiq(*args, **kwargs)
    
    # Manage: list_jobs, remove_job, pause_job, resume_job, start(), shutdown()
```

```python
from taskiq.brokers.inmemory_broker import InMemoryBroker
from taskiq.brokers.shared_broker import SharedBroker

class BrokerType(StrEnum):
    REDIS = "redis"
    RABBITMQ = "rabbitmq"
    KAFKA = "kafka"
    INMEMORY = "inmemory"
    UNKNOWN = "unknown"

class BrokerDetector:
    @staticmethod
    def detect(broker: AsyncBroker) -> BrokerType:
        try:
            from taskiq_redis.broker import RedisBroker
            if isinstance(broker, RedisBroker):
                return BrokerType.REDIS
        except ImportError: pass
        
        try:
            from taskiq_rabbit.broker import RabbitBroker
            if isinstance(broker, RabbitBroker):
                return BrokerType.RABBITMQ
        except ImportError: pass
        
        try:
            from taskiq_kafka.broker import KafkaBroker
            if isinstance(broker, KafkaBroker):
                return BrokerType.KAFKA
        except ImportError: pass
        
        if isinstance(broker, InMemoryBroker):
            return BrokerType.INMEMORY
        if isinstance(broker, SharedBroker):
            return BrokerDetector.detect(broker.broker)
        
        return BrokerType.UNKNOWN
```
